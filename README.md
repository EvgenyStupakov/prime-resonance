# Prime Resonance

Экспериментальный численный тест идеи обнаружения простых чисел с использованием **резонансных амплитуд**, построенных на основе **нетривиальных нулей функции Римана**.

**Author:** Evgeny Stupakov  
**Year:** 2026

---

## Идея

Каждому натуральному числу \( n \) сопоставляется комплексный сигнал

R(n) = | (Λ(n)+1) Σ exp(i(σ√n·γ + log(n+1) + φ(n))) |

где

- Λ(n) — функция фон Мангольдта  
- γ — мнимые части нулей дзета-функции Римана  
- σ = 0.5  
- φ(n) — поправка, зависящая от делителей числа  

В численном эксперименте наблюдается существенное различие резонансной амплитуды между **простыми** и **составными** числами.

---

## Результат эксперимента

Для N = 10000 получено:

```
Средний резонанс простых   : 56.050187
Средний резонанс составных : 6.378492
Соотношение prime/composite: 8.7874
```

Это показывает, что простые числа дают более сильный резонансный отклик.
